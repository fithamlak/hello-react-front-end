### Key Features <a name="key-features"></a>

- Setup react app with `npx create-react-app`
- Create the `App` component with react-router.
- Create the `Greeting` component that will display a greeting. Set it up as a route in  App component.
- Create a store, an action and a reducer that will connect to you API endpont to get the random greeting.
- Display the random greeting in `Greeting` component.
